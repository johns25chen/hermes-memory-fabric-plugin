## 3. What "P4 Proposal Scope" Means Here

P4 Proposal Scope means a possible future human-reviewed scope frame for drafting a later P4 proposal.

It is not the P4 proposal itself.

It is not P4 start.

It is not P4 approval.

It is not P4 authorization.

It is not P4 implementation.

It is not v7 start.

It is not product, MVP, or deployment work.

It may only define what a later proposal would be allowed to ask.

It cannot authorize implementation.

It cannot authorize productization.

It requires explicit human review later.

If any reader treats proposal scope as phase transition, approval, implementation permission, product work, v7 work, connector work, graph work, or durable memory writing permission, proposal-scope drafting must stop.

This document defines safe boundaries for a possible later proposal-scope document.

It prevents proposal language from becoming authorization.

It preserves sealed `v6.16.0`.
