## 17. Handoff and Status Reporting Contract

A handoff must report the instruction, task and route identity, baseline, fixed corpus, allowed and actual writes, validation results, smoke result, unresolved gaps, dispositions, authority boundaries, version state, and Git status. PASS means only documentary criteria passed. HOLD preserves unresolved evidence. STOP records prohibited scope. DEFER preserves the need for a later explicit decision. NEVER records a permanent prohibition.

Handoff transfers context, not approval, authorization, adoption, mutation, execution, deployment, launch, release, pricing, version, or successor authority.
