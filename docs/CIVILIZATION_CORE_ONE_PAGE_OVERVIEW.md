## 2. One-Minute Explanation

Civilization Core is a governance kernel for AI memory systems. It defines what can be remembered, what must remain candidate-only, what requires evidence, what requires review, what requires human approval, and what must never become automatic execution.

Subspace Memory System is the engineering carrier for that governance architecture.

Hermes Memory Fabric is the current repository and implementation carrier for the governed memory-fabric work.

Codex, OpenClaw, and Hermes are workflow or carrier tools. They are not the system identity.

`v6.16.0` is sealed as the Civilization Core Stable Kernel.

The post-terminal docs explain, package, audit, navigate, or discuss future possibilities. They do not reopen v6, authorize v7 implementation, authorize product implementation, create an MVP, or activate runtime behavior, dependencies, or adapters.
