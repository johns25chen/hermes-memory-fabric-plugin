## 6. Productization Boundary

Productization may be discussed after terminal closure, but it is outside the sealed v6 kernel.

Productization discussion must not:

- change the package version;
- modify source, tests, or scripts;
- add runtime bridges;
- add dependencies;
- add adapters;
- mutate memory graphs;
- create authorization or execution semantics;
- alter `v6.16.0` Stable Kernel meaning.

The v1-v6 Civilization Core roadmap is closed.
