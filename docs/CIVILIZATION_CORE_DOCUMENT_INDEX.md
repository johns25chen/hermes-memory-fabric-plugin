## 3. One-Minute Project Orientation

Civilization Core is a governance kernel for AI memory systems. It defines boundaries for memory, evidence, review, approval, execution, continuity, temporal reasoning, and source-methodology reasoning.

Subspace Memory System is the engineering carrier for the Civilization Core architecture. It names the memory-system shape that the governance kernel constrains.

Hermes Memory Fabric is the current repository and implementation carrier for the governed memory-fabric work.

Codex, OpenClaw, and Hermes are workflow or carrier tools. They are not the system identity. They may host documentation work, local validation, review, recall, or implementation carriers, but the Civilization Core identity is the governance architecture itself.

`v6.16.0` is sealed as the Civilization Core Stable Kernel.

The post-terminal docs are documentation / decision / roadmap / audit / navigation surfaces only. They explain, package, hand off, warn, compare, audit, or route readers. They do not reopen the v6 line and do not create implementation authority.
